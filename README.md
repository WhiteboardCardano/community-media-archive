# Cardano Community Media Archive

The **Cardano Community Media Archive** is an open-source initiative to preserve
and share design materials, media assets, and production files created for 
Cardano events worldwide.

This repository serves as a **structured index** of all event locations across 
multiple years. Each event folder contains:

- External download links  
- Metadata & previews  
- Credits  
- GitBook documentation references
